GitHub Instructions

How to upload a document to GitHub using the command line:

  1. Open a command line tool that you are using.
  2. Add the document that you want to upload:
    git add "index.html"
  3. Commit a document with a message:
    git commit -m "add new HTML file"
  4. Push the document to your repository:
    git push

Your document is uploaded to the repository.
